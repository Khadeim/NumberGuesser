# Number guesser Java console game - Khadeim Rahman

This is a console based number guessing game which runs in any terminal. It's simple and fun and features difficulty levels, a scoring system, replay option and local session leaderboard.

# Why I made it?

I made this game to utilise the foundational concepts of coding and learn basic Java syntax and concepts. I used things like conditionals, data structures and loops in a game which I can continue to scale as my knowledge progresses.

# How to run it?

1. Make sure you have **Java JDK** installed on your system.  
2. Save the code as: KhadNumberGuesser.java (to match the public class)
